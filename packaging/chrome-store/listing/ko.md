# 한국어 (ko)

**요약** (`_locales/ko`에서 자동):
사이드 패널에서 쓰는 60가지 이상의 개발자 도구. 모든 작업이 내 기기에서 실행됩니다. 업로드, 계정, 추적 없음.

**자세한 설명:**

```
QuiverKit은 60개가 넘는 개발자 도구를, 작업 중인 화면 옆에 계속 열어 둘 수 있는 사이드 패널에 담아 클릭 한 번으로 제공합니다.

아무 페이지에서나 텍스트를 선택하고 마우스 오른쪽 버튼을 눌러 "QuiverKit에서 열기"를 선택하세요 — 패널이 열리고 무엇을 선택했는지 파악해 알맞은 도구로 바로 안내합니다. 또는 툴바에서 패널을 열어 직접 도구를 고를 수도 있습니다.

모든 것이 사용자의 기기에서 실행됩니다. 업로드 없음, 계정 없음, 추적 없음 — 그리고 방문하는 페이지에 접근하지 않습니다. 이 확장 프로그램은 호스트 권한을 전혀 요청하지 않으며 탐색하는 사이트에서 아무것도 읽지 않습니다. 유일하게 다루는 데이터는 마우스 오른쪽 버튼 메뉴로 직접 보낸 텍스트뿐입니다.

포함된 도구:
• 인코딩 — Base64, Base32/58, URL, Punycode, Hex, JWT, QR 코드
• 포매팅 — JSON, JSON diff, JSONPath, SQL, XML, GraphQL, Markdown
• 텍스트 — 대소문자 변환, 정규식, diff, 정렬, 통계, 보이지 않는 문자, 암호
• 변환기 — CSV / TOML / YAML ↔ JSON, 타임스탬프, 시간대, 진법, 데이터 크기
• 디자인 — 색상 변환, 팔레트, 그라데이션, 그림자, CSS 단위, 베지어, SVG
• 암호 — 해시, HMAC, AES, 키 쌍, 인증서, TOTP, 체크섬
• 생성기 — UUID, 비밀번호, 로렘 입숨, 목업 데이터
• …그 외 다수

오픈 소스(MIT). https://quiverkit.dev 의 웹 앱과 Windows 및 Linux용 데스크톱 앱으로도 이용할 수 있습니다.
```
