**Ausführliche Beschreibung:**

```
QuiverKit bringt über 60 Entwickler-Tools mit einem Klick in ein Seitenpanel, das neben deiner Arbeit geöffnet bleibt.

Markiere Text auf einer beliebigen Seite, klicke mit der rechten Maustaste und wähle „In QuiverKit öffnen“ — das Panel öffnet sich, erkennt deine Auswahl und bringt dich direkt zum passenden Tool. Oder öffne das Panel über die Symbolleiste und wähle selbst ein Tool.

Alles läuft auf deinem eigenen Rechner. Kein Upload, kein Konto, kein Tracking — und kein Zugriff auf die Seiten, die du besuchst. Die Erweiterung fordert keine Host-Berechtigungen an und liest nichts von den Websites, die du aufrufst. Die einzigen Daten, die sie berührt, sind der Text, den du ihr bewusst über das Rechtsklick-Menü sendest.

Enthaltene Tools:
• Kodierung — Base64, Base32/58, URL, Punycode, Hex, JWT, QR-Codes
• Formatierung — JSON, JSON-Diff, JSONPath, SQL, XML, GraphQL, Markdown
• Text — Groß-/Kleinschreibung, Regex, Diff, Sortieren, Statistik, unsichtbare Zeichen, Chiffren
• Konverter — CSV / TOML / YAML ↔ JSON, Zeitstempel, Zeitzonen, Zahlensysteme, Datengrößen
• Design — Farbkonvertierung, Paletten, Verläufe, Schatten, CSS-Einheiten, Bézier, SVG
• Krypto — Hashes, HMAC, AES, Schlüsselpaare, Zertifikate, TOTP, Prüfsummen
• Generatoren — UUID, Passwörter, Lorem Ipsum, Testdaten
• …und mehr

Open Source (MIT). Auch als Web-App unter https://quiverkit.dev und als Desktop-App für Windows und Linux verfügbar.
```
