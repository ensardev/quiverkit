# 日本語 (ja)

**概要**（`_locales/ja` から自動）:
サイドパネルで使える60以上の開発者ツール。すべて手元の端末で動作します。アップロードなし、アカウントなし、追跡なし。

**詳細な説明:**

```
QuiverKit は 60 以上の開発者ツールを、作業中の画面の隣に開いたままにできるサイドパネルにまとめ、ワンクリックで使えるようにします。

任意のページでテキストを選択し、右クリックして「QuiverKit で開く」を選ぶと、パネルが開いて選択内容を判別し、最適なツールへ直接案内します。ツールバーからパネルを開いて、自分でツールを選ぶこともできます。

すべてがあなたの端末上で動作します。アップロードなし、アカウントなし、トラッキングなし — そして閲覧中のページへのアクセスもありません。この拡張機能はホスト権限を一切要求せず、閲覧するサイトから何も読み取りません。触れるのは、右クリックメニューから明示的に送ったテキストだけです。

収録ツール:
• エンコード — Base64、Base32/58、URL、Punycode、Hex、JWT、QR コード
• フォーマット — JSON、JSON 差分、JSONPath、SQL、XML、GraphQL、Markdown
• テキスト — 大文字小文字変換、正規表現、差分、並べ替え、統計、不可視文字、暗号
• 変換 — CSV / TOML / YAML ↔ JSON、タイムスタンプ、タイムゾーン、基数変換、データサイズ
• デザイン — 色変換、パレット、グラデーション、シャドウ、CSS 単位、ベジェ、SVG
• 暗号 — ハッシュ、HMAC、AES、鍵ペア、証明書、TOTP、チェックサム
• ジェネレーター — UUID、パスワード、ロレムイプサム、モックデータ
• …ほか

オープンソース（MIT）。https://quiverkit.dev のウェブアプリ、および Windows・Linux 向けのデスクトップアプリとしても利用できます。
```
