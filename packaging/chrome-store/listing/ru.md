# Русский (ru)

**Краткое описание** (автоматически из `_locales/ru`):
Более 60 инструментов разработчика в боковой панели. Всё работает на вашем компьютере: без загрузки и отслеживания.

**Подробное описание:**

```
QuiverKit помещает более 60 инструментов для разработчиков на расстояние одного клика — в боковую панель, которая остаётся открытой рядом с вашей работой.

Выделите текст на любой странице, щёлкните правой кнопкой и выберите «Открыть в QuiverKit» — панель откроется, определит, что вы выделили, и сразу направит вас к нужному инструменту. Или откройте панель с панели инструментов и выберите инструмент сами.

Всё работает на вашем компьютере. Никаких загрузок, аккаунтов и отслеживания — и никакого доступа к страницам, которые вы посещаете. Расширение не запрашивает host-разрешений и ничего не читает с сайтов, которые вы просматриваете. Единственные данные, которых оно касается, — это текст, который вы намеренно отправляете ему через контекстное меню.

Инструменты:
• Кодирование — Base64, Base32/58, URL, Punycode, Hex, JWT, QR-коды
• Форматирование — JSON, JSON diff, JSONPath, SQL, XML, GraphQL, Markdown
• Текст — регистр, regex, diff, сортировка, статистика, невидимые символы, шифры
• Конвертеры — CSV / TOML / YAML ↔ JSON, метки времени, часовые пояса, системы счисления, размеры данных
• Дизайн — конвертация цвета, палитры, градиенты, тени, единицы CSS, кривые Безье, SVG
• Криптография — хеши, HMAC, AES, пары ключей, сертификаты, TOTP, контрольные суммы
• Генераторы — UUID, пароли, lorem ipsum, тестовые данные
• …и другое

Открытый исходный код (MIT). Также доступно как веб-приложение на https://quiverkit.dev и как настольное приложение для Windows и Linux.
```
