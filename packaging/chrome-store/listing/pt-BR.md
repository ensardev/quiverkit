**Descrição detalhada:**

```
O QuiverKit coloca mais de 60 ferramentas para desenvolvedores a um clique, em um painel lateral que fica aberto ao lado do que você está fazendo.

Selecione um texto em qualquer página, clique com o botão direito e escolha "Abrir no QuiverKit" — o painel abre, descobre o que você selecionou e leva você direto para a ferramenta certa. Ou abra o painel pela barra de ferramentas e escolha você mesmo.

Tudo roda na sua própria máquina. Sem upload, sem conta, sem rastreamento — e sem acesso às páginas que você visita. A extensão não solicita permissões de host e não lê nada dos sites que você navega. Os únicos dados que ela toca são o texto que você envia deliberadamente pelo menu do botão direito.

Ferramentas incluídas:
• Codificação — Base64, Base32/58, URL, Punycode, Hex, JWT, códigos QR
• Formatação — JSON, JSON diff, JSONPath, SQL, XML, GraphQL, Markdown
• Texto — maiúsculas/minúsculas, regex, diff, ordenação, estatísticas, caracteres invisíveis, cifras
• Conversores — CSV / TOML / YAML ↔ JSON, timestamps, fusos horários, bases numéricas, tamanhos de dados
• Design — conversão de cores, paletas, gradientes, sombras, unidades CSS, bézier, SVG
• Cripto — hashes, HMAC, AES, pares de chaves, certificados, TOTP, checksums
• Geradores — UUID, senhas, lorem ipsum, dados de teste
• …e mais

Código aberto (MIT). Também disponível como app web em https://quiverkit.dev e como app de desktop para Windows e Linux.
```
