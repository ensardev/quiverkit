**Detailed description:**

```
QuiverKit puts 60+ developer tools one click away, in a side panel that stays open next to whatever you're working on.

Select text on any page, right-click, and choose "Open in QuiverKit" — the panel opens, works out what you selected, and sends you straight to the right tool. Or open the panel from the toolbar and pick a tool yourself.

Everything runs on your own machine. No upload, no account, no tracking — and no access to the pages you visit. The extension requests no host permissions and reads nothing from the sites you browse. The only data it ever touches is the text you explicitly send it from the right-click menu.

Tools include:
• Encoding — Base64, Base32/58, URL, Punycode, Hex, JWT, QR codes
• Formatting — JSON, JSON diff, JSONPath, SQL, XML, GraphQL, Markdown
• Text — case convert, regex, diff, sort, stats, invisible characters, ciphers
• Converters — CSV / TOML / YAML ↔ JSON, timestamps, time zones, number bases, data sizes
• Design — colour convert, palettes, gradients, shadows, CSS units, bézier, SVG
• Crypto — hashes, HMAC, AES, key pairs, certificates, TOTP, checksums
• Generators — UUID, passwords, lorem ipsum, mock data
• …and more

Open source (MIT). Also available as a web app at https://quiverkit.dev and as a desktop app for Windows and Linux.
```
