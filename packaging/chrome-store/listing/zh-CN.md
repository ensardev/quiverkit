**详细说明：**

```
QuiverKit 将 60 多种开发者工具放进一个侧边栏，一键即用，并始终停留在你工作内容的旁边。

在任意页面选中文本，右键选择“用 QuiverKit 打开”——侧边栏随即打开，识别你选中的内容，并直接把你带到合适的工具。你也可以从工具栏打开侧边栏，自己挑选工具。

一切都在你自己的设备上运行。无需上传、无需账户、没有跟踪——也不会访问你浏览的页面。该扩展不请求任何主机权限，也不会读取你浏览的任何网站。它唯一接触的数据，就是你通过右键菜单主动发送的文本。

包含的工具：
• 编码 — Base64、Base32/58、URL、Punycode、Hex、JWT、二维码
• 格式化 — JSON、JSON 对比、JSONPath、SQL、XML、GraphQL、Markdown
• 文本 — 大小写转换、正则、对比、排序、统计、不可见字符、古典密码
• 转换器 — CSV / TOML / YAML ↔ JSON、时间戳、时区、进制、数据大小
• 设计 — 颜色转换、调色板、渐变、阴影、CSS 单位、贝塞尔曲线、SVG
• 加密 — 哈希、HMAC、AES、密钥对、证书、TOTP、校验和
• 生成器 — UUID、密码、乱数假文、模拟数据
• ……以及更多

开源（MIT）。也可作为网页应用（https://quiverkit.dev）以及 Windows 和 Linux 桌面应用使用。
```
